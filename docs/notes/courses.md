# 吴恩达课程全览

> Andrew Ng 课程汇总，按学习顺序排列

## 入门理论类

| 课程 | 平台 | 说明 |
|:----|:----|:----|
| AI For Everyone | Coursera | AI通识入门，适合所有人 |
| Machine Learning Specialization | Coursera | 经典ML课，2022年重制版 |
| Deep Learning Specialization | Coursera | 深度学习核心课 |

## AI应用类

| 课程 | 时长 | 说明 |
|:----|:---:|:----|
| LangChain for LLM App Development | ~3h | LLM应用开发 |
| ChatGPT Prompt Engineering for Developers | ~2h | 提示词工程 |
| LangChain RAG | ~2h | 检索增强生成 |

## 学习建议

1. 先看 AI For Everyone，建立整体认知
2. 再看 Machine Learning Specialization，打基础
3. 然后看 Prompt Engineering 和 LangChain，直接动手
4. 最后看 Deep Learning Specialization，深入原理
